Optional track alignment

            

              
              
            

            

              
              
            

            

              
              
            

            

              
              
            

            

              
              
            

            

              
              
            

            
Used for AudD, AcoustID, and Songrec window scans. AudD windows are capped to 12 seconds.

            
Sample count is automatic per episode (based on track count and duration).

          

          

            
FFmpeg cue landmark filters

            

              
              
            

            

              
              
            

            

              
              
            

            
These filters estimate chapter/break landmarks only. They do not identify songs.

          

          

            
Download rules

            

              
              
            

            
Set to 0 to keep everything. When above 0, older files for the same program are pruned after new downloads finish.

            

              
              
            

            
Set to 0 to disable age pruning.

            

              
              
            

          

          

            
Tag cleanup

            

              
              
            

            
Removes duplicate program prefixes in episode titles and keeps host/genre tags tidy before writing artwork, chapters, and metadata.
